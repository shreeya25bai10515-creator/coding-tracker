This is a simple app to track how many coding problems you solve daily.

## How It Works

- You can **add** your daily progress.
- You can **view** your past progress and see your current streak.
- You can **exit** the app anytime.

## How to Run

1. Make sure you have **Python installed**.
2. Open terminal / command prompt.
3. Go to the folder with `main.py`.
4. Run the app:

```bash
python main.py
````

5. Type **add** to add problems solved, **view** to see progress, or **exit** to quit.

## Example

```
What do you want to do? add
How many problems did you solve today? 3
Progress saved!

What do you want to do? view

Your Progress:
Day 1: 3 problems
🔥 Current Streak: 1 days
```

I wanted a simple way to **stay consistent with coding** and track my daily progress. It helps me stay motivated.
